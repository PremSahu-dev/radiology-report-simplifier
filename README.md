# AI Radiology Report Simplifier

## Overview
AI Radiology Report Simplifier is a college team project developed to simplify radiology diagnosis using Artificial Intelligence. The system allows users to upload medical images and generates AI-assisted predictions along with simplified radiology reports.

## Features
- User Authentication
- Patient Management
- Medical Image Upload
- AI Prediction Integration
- Simplified Radiology Reports
- Report History
- Dashboard

## Technologies Used

### Frontend

### Backend

### AI
- PyTorch
- ResNet18
- Grad-CAM
