# Sample DICOM Files

This folder contains a small set of sample DICOM images used for testing the AI module.

Files included:
- imgMR1.dcm
- imgMR2.dcm
- imgCT1.dcm
- imgCT2.dcm
- imgXR1.dcm
- imgXR2.dcm

These files are intended only for development and testing.

Large medical imaging datasets should not be committed to GitHub.
